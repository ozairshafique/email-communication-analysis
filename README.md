# 📧 Email Communication Analysis

## 📌 Overview

In this project, I worked with real email data to explore communication patterns and understand how emails are distributed over time. The goal was to take raw and unstructured email data and turn it into meaningful insights using Python.

The analysis focuses on identifying when emails are most frequent, who the main contacts are, and how communication behavior changes across different time periods.

---

## 🎯 Problem Statement

Email data contains a lot of hidden information, but it is not easy to interpret in its raw form. By analyzing this data, we can better understand communication habits, detect patterns, and gain insights that can be useful for improving productivity and organization.

---

## 🚀 Approach

The project follows a step-by-step data analysis pipeline:

- Extracted email data from `.mbox` files
- Converted raw data into a structured CSV format
- Cleaned and processed the dataset
- Extracted useful features such as:
  - Sender email addresses
  - Date and time information
  - Day of the week and hourly activity
- Performed exploratory data analysis (EDA)
- Visualized results using charts and word clouds

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- Regular Expressions (re)

---

## 📊 Key Features

- Analysis of email frequency over time
- Identification of most frequent senders and recipients
- Time-based analysis (hour, day, month)
- Visualization of communication trends
- Subject analysis using word cloud

---

## 📈 Insights

Some observations from the analysis:

- Email activity is higher during weekdays compared to weekends
- Most emails are received during typical working hours
- A few senders contribute to a large portion of total emails
- Subject lines reveal recurring topics and patterns
